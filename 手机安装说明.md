# 连连看 PWA 手机安装说明

这个目录已经是 PWA 版本，包含：

- `manifest.webmanifest`
- `sw.js`
- `icons/icon-192.png`
- `icons/icon-512.png`

手机安装到桌面的关键条件：

1. 用 HTTPS 地址访问这个游戏。手机上的普通 HTTP 地址通常不能注册 service worker，也就不能作为 PWA 安装。
2. Android Chrome：打开 HTTPS 地址后，菜单里选择“安装应用”或“添加到主屏幕”。
3. iPhone Safari：打开 HTTPS 地址后，点分享按钮，再选择“添加到主屏幕”。

本机测试可以用：

```bash
python3 -m http.server 4173
```

然后在电脑浏览器打开：

```text
http://localhost:4173/
```

注意：`localhost` 只适合电脑本机测试。要安装到手机桌面，建议把整个目录部署到 GitHub Pages、Vercel、Netlify 或其他 HTTPS 静态网站服务。
