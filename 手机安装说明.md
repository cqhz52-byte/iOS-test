# 连连看 PWA 手机安装说明

这个仓库已经是 PWA 版本，包含：

- `manifest.webmanifest`
- `sw.js`
- `icons/icon.svg`

手机安装到桌面的关键条件：

1. 用 HTTPS 地址访问这个游戏。GitHub Pages 地址满足这个条件。
2. Android Chrome：打开 HTTPS 地址后，菜单里选择“安装应用”或“添加到主屏幕”。
3. iPhone Safari：打开 HTTPS 地址后，点分享按钮，再选择“添加到主屏幕”。
